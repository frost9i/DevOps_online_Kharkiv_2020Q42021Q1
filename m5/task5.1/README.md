# Task 5.1 - Linux essentials  
  
***
  
## Part 1. - #  
  
Being logged it as 'root' user can perform any system commands without using \>sudo modifier.  
Changing root password:
```
passwd  
```
Changing <user> password as root:  
```
passwd <user>  
```
